# DucksBedtimeBot

TelegramBot to remind us to bring ducks to bed

```
export TELEGRAM_BOT_TOKEN=[TOKEN]
```

## Features

* [ ] `/start` welcome message
  * [ ] Verify userID with JSON file 
  * [ ] Command parser
  * [ ] Command actions 
* [ ] send an alarm with clockwerk
* [ ] `/score` in json file
* [ ] Bot tui dashboard
